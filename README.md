# 🍔 Edzy School Canteen – Frontend Prototype

A **React-based prototype** for a school canteen digital ordering system.  
Students can explore snacks, place orders, and track their spending history.

This project was built as part of a **Frontend Screening Task**, focusing on clean UI, modular components, state management, and good UX practices.

---

## 🚀 Features

### 🥪 Snacks Page
- Displays a list of available snacks
- Each snack shows:
  - Image
  - Name
  - Price
- Quantity selector (1–5)
- **Order button opens a modal** to choose student and quantity

### 👩‍🎓 Students Page
- Displays all students in card format
- Each student card shows:
  - Name
  - Referral Code
  - Total Amount Spent
- Button to **view student details**

### ➕ Add Student
- Modal form to create a new student
- User enters:
  - Student Name
  - Image URL (optional)
- **Referral Code is generated automatically**

### 🍽️ Category Filtering
- Snacks can be filtered by category using the **Explore Menu section**

### 🎨 UI/UX Features
- Responsive layout
- Modal-based interactions
- Smooth hover effects
- Clean component structure

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/AnkitaMaurya-2908/school-canteen-react.git
```

### 2️⃣ Navigate to the project

```bash
cd school-canteen
```

### 3️⃣ Install dependencies

```bash
npm install
```

### 4️⃣ Run the development server

```bash
npm run dev
```
